CAGED Guitar System

The CAGED guitar system is a method for learning the guitar fretboard that uses five basic chord shapes: C, A, G, E, and D. These shapes can be moved up and down the neck to play the same chord in different positions.

Understanding the CAGED Shapes

How to Use the CAGED System

Each shape corresponds to a chord and can be moved up and down the neck to form other chords of the same type. For example, the C shape can be moved up the neck to play a D chord, an E chord, and so on.

Practice Exercises

To get comfortable with the CAGED system, try these practice exercises:

  • Move each shape up and down the neck and play different chords.
  • Combine shapes to play chord progressions.
  • Identify the shapes in songs you know.
